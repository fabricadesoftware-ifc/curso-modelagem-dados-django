# Visão geral do projeto a ser modelado

Vamos modelar um sistema para gerenciamento de filmes. O sistema deve permitir o cadastro de filmes, com os seguintes dados:

* Título
* Diretor
* Ano de lançamento
* Gênero
* Duração
* Sinopse
* Atores
* País de origem
* Idioma original
* Produtora
